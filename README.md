# 🦋 LumeMoth | Universal Project Architect

![Version](https://img.shields.io/badge/version-3.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

**LumeMoth** is an intelligent, language-agnostic CLI tool designed to audit your codebase. It identifies logic errors, technical debt, and "code smells" while offering interactive **Auto-Fix** capabilities.

---

## 🚀 Features

- **🔍 Universal Scanning**: Supports `.ts`, `.js`, `.py`, `.java`, `.rs`, `.cpp`, `.cs`, and more.
- **🛡️ Deep Logic Analysis**: Detects silenced errors, forgotten debug logs, and unresolved `TODO` markers.
- **🛠️ Interactive Auto-Fix**: Choose to patch your code directly from the terminal.
- **🎫 GitHub Ticket Architect**: Generates professional Markdown reports ready for copy-pasting into GitHub Issues.
